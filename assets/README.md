# IPCC Climate Regions (Shapefile)
This folder contains the vector data used to define the 5 climatic sub-regions (ROI) for the study.

## File Description
*   **IPCC_5_roi_shapefile.shp**: Main geometry file.
*   **IPCC_5_roi_shapefile.dbf**: Attribute data (Region names: EAF, WAF, SAF, SAH, MED).
*   **IPCC_5_roi_shapefile.prj**: Projection information (EPSG:4326).

## Usage in GEE
To use this in Google Earth Engine:
1.  Download all files in this folder.
2.  Go to the GEE Code Editor -> Assets -> New -> Shapefile.
3.  Upload the .shp, .dbf, .shx, and .prj files together.
